⚡ QUICK STEPS

Step 1: Apply for Developer Account (10 min - DO FIRST!)
─────────────────────────────────────────────────────────
1. Go to: https://play.google.com/console/signup
2. Pay $25
3. Wait 1-2 days for approval

Step 2: Code Changes (6 hours with AI help)
────────────────────────────────────────────
1. Rename keystore: wellness-buddy-keystore.jks → wellness-valley-keystore.jks
2. Update package ID in all files
3. Update "Wellness Buddy" to "Wellness Valley" everywhere

Step 3: Firebase (15 min)
──────────────────────────
1. Firebase Console → Add Android app
2. Package: com.wellnessvalley.app
3. Add SHA-1 from keystore
4. Download google-services.json

Step 4: Graphics (2-3 hours)
────────────────────────────
1. Canva AI → Create feature graphic with "Wellness Valley"
2. Take 4-8 screenshots from app

Step 5: Build & Test (2 hours)
───────────────────────────────
1. Build release bundle: gradlew.bat bundleRelease
2. Test on device
3. Fix any bugs

Step 6: Closed Testing Setup (2-3 hours)
─────────────────────────────────────────
1. Create app in Play Console
2. Upload graphics + AAB to CLOSED TESTING track
3. Fill store listing (text already prepared)
4. Create tester list: Add 20 email addresses (friends/family/colleagues)
5. Share testing link with testers
6. Wait 14 days (mandatory for new organizational accounts)
7. After 14 days: Promote to Production and submit!
