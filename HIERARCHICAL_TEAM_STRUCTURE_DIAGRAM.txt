┌──────────────────────────────────────────────────────────────────────────┐
│                          DATA FLOW EXAMPLE                               │
└──────────────────────────────────────────────────────────────────────────┘

1. USER CLICKS "All Teams" TAB
   ↓
2. React state changes: setActiveTab('allTeams')
   ↓
3. useEffect triggers: loadHierarchyData()
   ↓
4. Service called: teamHierarchyService.getTeamHierarchy(userId)
   ↓
5. HTTP GET: /api/coach/team-hierarchy?coachId=123
   ↓
6. Backend queries: SELECT * FROM team_table WHERE Status='Active'
   ↓
7. Backend builds tree: Recursive hierarchy construction
   ↓
8. Backend returns JSON:
   {
     userId: 123,
     userName: "Main Coach",
     teamMembers: [
       { userId: 456, userName: "Bob", teamMembers: [...] }
     ]
   }
   ↓
9. Service receives data, stores in state: setHierarchyData(data)
   ↓
10. Component renders: <HierarchicalTeamView hierarchy={data} />
    ↓
11. Recursive rendering: Each TeamNode renders its children
    ↓
12. USER SEES: Interactive tree with expand/collapse

┌──────────────────────────────────────────────────────────────────────────┐
│                       INTERACTION PATTERNS                               │
└──────────────────────────────────────────────────────────────────────────┘

Pattern 1: EXPAND NODE
User clicks chevron (›) → 
  TeamNode.handleToggle() → 
    onToggleExpand(nodeId) → 
      setExpandedNodes(add nodeId) → 
        Re-render with children visible

Pattern 2: COLLAPSE NODE
User clicks chevron (∨) → 
  TeamNode.handleToggle() → 
    onToggleExpand(nodeId) → 
      setExpandedNodes(remove nodeId) → 
        Re-render with children hidden

Pattern 3: CLICK NODE
User clicks node card → 
  onNodeClick(node) → 
    setActiveTab('myTeam') → 
      setTeamFilter(node.userId) → 
        Filter flat list by coach

Pattern 4: EXPAND ALL
User clicks [Expand All] → 
  collectIds(hierarchy) → 
    setExpandedNodes(allIds) → 
      Re-render all nodes expanded

Pattern 5: SWITCH TAB
User clicks [All Teams] → 
  setActiveTab('allTeams') → 
    useEffect triggers → 
      loadHierarchyData() → 
        API call → 
          Display hierarchy
